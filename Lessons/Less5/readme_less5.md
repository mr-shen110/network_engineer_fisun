* Узнал что "ip helper-address" в cisco-router может ретранслитить 8 служб: 
** port 37: Time
** port 49: TACACS
** port 53: DNS
** port 67: DHCP/BOOTP server 
** port 68: DHCP/BOOTP client 
** port 69: TFTP
** port 137: NetBIOS name service 
** port 138: NetBIOS datagram service
